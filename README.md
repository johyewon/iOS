		* assert
			- 특정 조건을 체크하고 조건이 성립되지 않으면 메시지를 출력하게 할 수 있는 함수

		* guard - 방어구문
			- 검사 후 다음에 오는 코드를 실행할지 말지 결정하는 것
			- guard 문에 주어진 조건문이 거짓일 때 실행
			guard 조건 else {
				return or throw or break 를 통해 코드 실행 않도록 함
			}
